## Shadcraft Free — shadcn/ui Registry

**Shadcraft Free** is a free, open‑source shadcn/ui registry that ships polished components and marketing blocks built to production standards.

- **What it is**: A set of copy‑paste ready components and blocks designed to work seamlessly with `shadcn/ui`, Tailwind CSS, and modern React projects.
- **Use cases**: Landing pages and marketing sites where you want high‑quality building blocks without starting from scratch.
- **How to use**: Add items to your project using the `shadcn` CLI. @shadcraft namespace is automatically resolved and configured in your `components.json`.

Shadcraft is listed in the official shadcn/ui Registry Directory.
